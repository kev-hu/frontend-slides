SECTION 02 — ARCHITECTURE
// THREE PRIMITIVES

One API surface, every interaction.

act()

Perform any UI action from a plain-English instruction. Clicks, typing, and navigation resolve to stable steps.

// MUTATION

extract()

Pull typed, schema-validated data out of any page. Returns structured JSON you can trust downstream.

// READ

observe()

Plan ahead — preview the candidate actions on a page before committing, for safe agentic loops.

// PLAN
// resilient by intent, not by selector
await page.act("click the 'Start free trial' button");
