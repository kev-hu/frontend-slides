## Visual Snapshot

A charming macOS-inspired glassmorphism system that turns a deck into a desktop. Every block of content lives inside a floating frosted-glass **app window** with a traffic-light titlebar (red/amber/green dots) and a centered monospace filename pill, floating on a **warm-beige desktop wallpaper** lit by three soft radial glows — orange top-left, purple top-right, green bottom-center. A glassy menubar (with the canonical 9:41 clock) pins the top; a Finder-style explorer sidebar with an orange-highlighted active row handles navigation; glass widget cards, a floating AI-coach chat bubble, and a bottom dock give it a lived-in, product-minded personality.

Type is a single OS superfamily: **Geist** for everything human (display/UI/body, tight negative tracking at large sizes) and **Geist Mono** for everything the machine emits (filename pills, `//` eyebrows, badges, file paths, code snippets, counters). One friendly **orange (#F9A526)** is the only brand/CTA color (with a glowing button shadow); a green/purple/red/amber status spectrum is quarantined to badges, status dots, traffic lights, and the wallpaper glows. Generously rounded corners and soft diffuse shadows make it the deliberate inverse of a flat or brutalist deck — light-mode, warm, and precise.
